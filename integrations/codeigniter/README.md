# CodeIgniter Integration

This package will connect the GenReport runtime to CodeIgniter applications.

## Purpose

The CodeIgniter integration should expose runtime capabilities in a way that fits CodeIgniter conventions.

## Important Boundary

This package is not the runtime itself. It should connect the runtime to the framework, not replace runtime behavior.

## Future Package Name

`genreport/codeigniter`

## Future Install Command

```bash
composer require genreport/codeigniter
```
