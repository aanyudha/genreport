# Django Integration

This package will connect the GenReport runtime to Django applications.

## Purpose

The Django integration should make it easier to attach runtime execution to Django views, services, and configuration.

## Important Boundary

This package is not the runtime itself. It should connect the runtime to the framework, not replace runtime behavior.

## Future Package Name

`genreport-django`

## Future Install Command

```bash
pip install genreport-django
```
