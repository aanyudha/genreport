# Express Integration

This package will connect the GenReport runtime to Express applications.

## Purpose

The Express integration should help Node.js applications expose GenReport execution through HTTP or service workflows.

## Important Boundary

This package is not the runtime itself. It should connect the runtime to the framework, not replace runtime behavior.

## Future Package Name

`@genreport/express`

## Future Install Command

```bash
npm install @genreport/express
```
