# Laravel Integration

This package will connect the GenReport runtime to Laravel applications.

## Purpose

The Laravel integration should provide framework-friendly wiring, configuration, and delivery helpers for Laravel users.

## Important Boundary

This package is not the runtime itself. It should connect the runtime to the framework, not replace runtime behavior.

## Future Package Name

`genreport/laravel`

## Future Install Command

```bash
composer require genreport/laravel
```
