# Spring Boot Integration

This package will connect the GenReport runtime to Spring Boot applications.

## Purpose

The Spring Boot integration should adapt runtime behavior to common Spring Boot application patterns.

## Important Boundary

This package is not the runtime itself. It should connect the runtime to the framework, not replace runtime behavior.

## Future Package Name

`io.genreport:genreport-springboot`
