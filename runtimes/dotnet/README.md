# .NET Runtime

This package will be the .NET runtime for GenReport.

## Purpose

The .NET runtime is intended to execute GenReport Definitions within the .NET ecosystem.

## Future Package Name

`GenReport.Runtime`

## Future Install Command

```bash
# NuGet package planned
```

## Responsibility

The runtime is expected to:

- load schema definitions
- validate schema definitions
- execute reports
- render or hand off export-ready results
