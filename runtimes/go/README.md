# Go Runtime

This package will be the Go runtime for GenReport.

## Purpose

The Go runtime is intended to execute GenReport Definitions in Go services and tooling.

## Future Package Name

`github.com/loxar/genreport-go` or `github.com/genreport/runtime-go`

## Future Install Command

```bash
# Go module path planned
```

## Responsibility

The runtime is expected to:

- load schema definitions
- validate schema definitions
- execute reports
- render or hand off export-ready results
