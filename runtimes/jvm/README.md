# JVM Runtime

This package will be the JVM runtime for GenReport.

## Purpose

The JVM runtime is intended to support Java and Kotlin ecosystem usage of GenReport Definitions.

## Future Package Name

`io.genreport:genreport-runtime`

## Future Install Command

```bash
# Maven / Gradle coordinates planned
```

## Responsibility

The runtime is expected to:

- load schema definitions
- validate schema definitions
- execute reports
- render or hand off export-ready results
