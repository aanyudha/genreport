# PHP Runtime

This package will be the PHP runtime for GenReport.

## Purpose

The PHP runtime is intended to execute GenReport Definitions directly within PHP applications and services.

## Future Package Name

`genreport/runtime`

## Future Install Command

```bash
composer require genreport/runtime
```

## Responsibility

The runtime is expected to:

- load schema definitions
- validate schema definitions
- execute reports
- render or hand off export-ready results
