# Python Runtime

This package will be the Python runtime for GenReport.

## Purpose

The Python runtime is intended to execute GenReport Definitions directly in Python-based systems and automation flows.

## Future Package Name

`genreport-runtime`

## Future Install Command

```bash
pip install genreport-runtime
```

## Responsibility

The runtime is expected to:

- load schema definitions
- validate schema definitions
- execute reports
- render or hand off export-ready results
